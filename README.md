# Infinite Mario Bros 2:

Click on the exe files if u want to play it...

⚠IMPORTANT!!!⚠ Controls:

```
If you are in a level:
Right and left keys: Move left or right.
Z: Jump.
X: Shoot.

If you are on the map:
Arrow keys: Move in the map. Collide a tree, a dick, a palm or a piranha plant to enter a level.
A: Open game.
S: Save game.
```

... or open it with game maker 8.0 pro if u want to edit it...
